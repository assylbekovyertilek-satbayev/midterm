## Жобаның мақсаты
Бұл midterm жобасы студенттің техникалық құжаттаманы әзірлеу, құрылымдау, стандарттау, басқару және аудит жүргізу дағдыларын қалыптастыруға бағытталған

## Жоба міндеттері
Осы мақсатқа жету үшін жоба барысында келесі нақты міндеттер орындалады

Ғылыми мәтінді жүйелі түрде құрылымдау

Халықаралық және мемлекеттік стандарттарға сәйкес құжат әзірлеу

Ақпараттық жүйенің техникалық талаптарын (архитектура, модульдер) рәсімдеу

Құжаттаманы басқару құралдарын (Git, GitHub) тәжірибеде қолдану

Әзірленген құжаттың сапасына чек-лист арқылы кәсіби талдау (аудит) жүргізу

## Қолданылған технологиялар тізімі (Технологиялық стек) 

Фронтенд: React.js (Пайдаланушы интерфейсі)

Бэкенд: Node.js және Express.js (Қолданбалы логика)

Деректер базасы: PostgreSQL (Ақпаратты сақтау)

Инфрақұрылым: Docker (Контейнерлеу)

Құжаттама және басқару: Microsoft Word (стильдер мен форматтау), Git және GitHub (нұсқаларды басқару)

## Құжаттама аудитінің чек-листі 
[x] Кіріспе бөлімі жазылды (өзектілігі, мақсаты, міндеттері)

[x] Теориялық бөлім дайындалды (10 ғылыми дереккөз, 10-15 негізгі термин)

[x] Жобалық бөлім рәсімделді (жүйе архитектурасы, функционалдық модульдер) 

[x] Құжатта кемінде 1 кесте және 1 диаграмма/схема бар 

[x] Құжат стильдер арқылы форматталды және автоматты мазмұн құрылды 

[x] Git репозиторийі ашылып, кемінде 6 commit жасалды

## Құжатты жетілдіру (Refactoring) жоспары 

Сілтемелерді динамикалық басқару: Кестелер мен диаграммаларға арналған Word кросс-сілтемелерін (cross-references) толық автоматтандыру.

## Қысқаша рефлексия
Бұл жоба барысында мен техникалық құжаттаманы халықаралық стандарттарға сай құрылымдауды, автоматты мазмұн мен стильдерді қолдануды және Git жүйесімен жұмыс істеуді үйрендім. Аудит жүргізу мен рефакторинг жоспарын құру арқылы жоба сапасын бағалау дағдыларын қалыптастырдым.

Терминологиялық бірізділік: Глоссарийдегі терминдердің құжат бойына бірдей қолданылуын қосымша тексеру.

Визуалды иерархияны жақсарту: Абзацтар мен кестелерді мемлекеттік стандарттарға (ҚР СТ) қатаң сәйкестендіру.

Библиографияны автоматтандыру: Пайдаланылған әдебиеттер тізімін басқару үшін Mendeley немесе Zotero бағдарламаларына көшу.
