## Жобаның мақсаты
[cite_start]Бұл midterm жобасы студенттің техникалық құжаттаманы әзірлеу, құрылымдау, стандарттау, басқару және аудит жүргізу дағдыларын қалыптастыруға бағытталған[cite: 3, 4].

## Жоба міндеттері
[cite_start]Осы мақсатқа жету үшін жоба барысында келесі нақты міндеттер орындалады[cite: 5]:
* [cite_start]Ғылыми мәтінді жүйелі түрде құрылымдау[cite: 6];
* [cite_start]Халықаралық және мемлекеттік стандарттарға сәйкес құжат әзірлеу[cite: 7];
* [cite_start]Ақпараттық жүйенің техникалық талаптарын (архитектура, модульдер) рәсімдеу[cite: 8];
* [cite_start]Құжаттаманы басқару құралдарын (Git, GitHub) тәжірибеде қолдану[cite: 9];
* [cite_start]Әзірленген құжаттың сапасына чек-лист арқылы кәсіби талдау (аудит) жүргізу[cite: 10].
